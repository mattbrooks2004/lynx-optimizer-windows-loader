## Troubleshooting

- If the loader will not open, make sure your system supports Windows 11 or Windows 10.
- If files look incomplete, download the package again and extract it once more.
- If update checks do not succeed, confirm your network connection or use the manual download path.
- If a profile does not apply correctly, verify that you have permission to access the local files and folders used by the loader.
- If you need to reverse changes, use the safe rollback option before trying a different profile.
- If the tool is being launched from a restricted directory, move it to a normal user-writable folder and start it again.
